Orange Juice 是一个在线评测系统的 API 服务器。

项目的主要任务有：

\begin{itemize}
    \item 对涉及的实体进行建模
    
    \begin{itemize}
        \item 用户 User
        \item 题目 Problem
        \item 题单 Problem List
        \item 提交 Submission
    \end{itemize}
    
    由陈铮负责，柳亚辉、徐明希、郭家梁参与讨论。
    
    \item 编写 API 服务器核心代码与测试
    
    \begin{itemize}
        \item 用户
        \item 题目
        \item 题单
        \item 提交
    \end{itemize}
    
    陈铮、柳亚辉、郭家梁参与编写核心代码。
    
    陈铮、柳亚辉、徐明希、郭家梁、欧阳鹏程、曹思璐参与编写测试代码。
